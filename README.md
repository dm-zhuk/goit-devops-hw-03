# DevOps Task: Automated Environment Setup

This repository contains a Bash script to automate the installation of essential development tools on an Ubuntu system.

## Tools Installed

* **Docker**
* **Docker Compose**
* **Python 3.9+**
* **Django**

## How to Use

1. **Make the script executable:**
   ```bash
   chmod u+x install_dev_tools.sh
2. **Run the installation on an Ubuntu system:**
   ```bash
   ./install_dev_tools.sh
